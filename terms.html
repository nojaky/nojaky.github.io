이용 안내

1. 서비스 목적

로스터현 웹 도구는 카페 창업, 운영, 메뉴 원가, 커피 가격, 추출 기록을 참고하기 위한 무료 정보 도구입니다.

2. 계산 결과 안내

모든 계산 결과는 참고용입니다. 실제 창업 비용, 매출, 원가, 수익, 거래 가격은 상권, 계약 조건, 운영 방식, 세금, 재료비, 환율 등에 따라 달라질 수 있습니다.

3. 데이터 안내

일부 도구는 공개 데이터와 사용자가 입력한 값을 바탕으로 결과를 계산합니다. 데이터 지연, 환산 방식, 입력값 오류에 따라 실제 결과와 차이가 날 수 있습니다.

4. 외부 링크

이 사이트는 GitHub Pages 및 외부 도구 페이지로 연결될 수 있습니다. 외부 사이트 이용 시 해당 사이트의 정책을 함께 확인해 주세요.
