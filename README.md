# 로스터현 커피 도구 통합 홈페이지

`nojaky.github.io` 저장소의 GitHub Pages용 홈페이지입니다.

## 업로드

다음 파일을 저장소 최상위에 업로드합니다.

- `index.html`
- `styles.css`
- `privacy-policy.html`

업로드 후 `Settings` > `Pages`에서 다음과 같이 설정합니다.

```text
Source: Deploy from a branch
Branch: main
Folder: /(root)
```

공개 주소:

```text
https://nojaky.github.io
```
