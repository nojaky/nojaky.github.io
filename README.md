# 커피볶는 개발자 로스터현 웹 도구

로스터현이 직접 만든 카페 운영·창업·커피 관련 무료 웹 도구를 소개하는 GitHub Pages 메인 사이트입니다.

## 주요 파일
- `index.html`: 웹 도구 목록과 메인 소개
- `about.html`: 운영자 및 사이트 소개
- `contact.html`: 문의 안내
- `privacy-policy.html`: 개인정보처리방침
- `terms.html`: 이용 안내
- `sitemap.xml`, `sitemap.txt`, `robots.txt`: 검색엔진 등록 파일

블로그 콘텐츠는 티스토리에서 별도로 운영하며, 이 저장소는 웹 도구 안내에 집중합니다.
