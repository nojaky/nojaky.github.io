로스터현 블로그 글쓰기 에디터

새 글을 작성하거나 기존 블로그 HTML 파일을 불러와 수정한 뒤 다시 HTML 파일로 저장하는 도구입니다. 서버 저장 기능은 없으며, 저장한 파일을 GitHub 저장소의 /blog/ 폴더에 직접 업로드해서 반영합니다.

  • 기존 글을 불러오면 제목, 파일명, 카테고리, 요약, 작성일, 본문이 자동으로 채워집니다.
  • 본문은 다시 수정하기 쉽게 ## 소제목, !! 굵은 줄, 빈줄 형식으로 복원됩니다.
  • 이 페이지는 관리자 도구이므로 검색 노출을 막기 위해 noindex가 적용되어 있습니다.

글 작성 / 수정

영문 소문자, 숫자, 하이픈 권장. 저장 파일명은 입력값.html입니다. 기존 글을 불러오면 기존 파일명이 자동으로 들어갑니다.

## 큰 소제목 ### 작은 소제목 !! 줄 전체 굵게 - 목록 항목

빈 줄을 2줄 이상 넣으면 글 안에 추가 여백이 들어갑니다. 기존 글 불러오기 기능은 이 문법으로 본문을 최대한 복원합니다.
